Informativa sulla Privacy

L’applicazione non raccoglie né memorizza dati personali degli utenti.

Non vengono utilizzati cookie propri né cookie di tracciamento o profilazione.

Non esiste alcun sistema di backend o raccolta dati lato server: tutte le funzionalità operano direttamente nel browser dell’utente.

Alcune informazioni (come preferenze o contenuti generati) possono essere salvate localmente tramite localStorage; questi dati restano sul dispositivo dell’utente e non vengono inviati a server esterni.

L’app può utilizzare servizi di terze parti (es. API pubbliche, Google Fonts, servizi di IA) che potrebbero registrare automaticamente informazioni tecniche come l’indirizzo IP, secondo le loro politiche sulla privacy.

L’informativa può essere aggiornata in qualsiasi momento per riflettere modifiche al funzionamento dell’app.
